# Matriz curricular da rede municipal de Mendes

Esta pasta guarda a base curricular local atualmente versionada no projeto.

## Objetivo
- registrar documentalmente a matriz curricular municipal usada como referencia local;
- permitir preenchimento da coluna `Campo para matriz da rede` em `docs/CONVERGENCIA_AVALIACOES_BNCC_REDE.md`;
- preparar substituicao futura por outra rede, sem perder o historico de `Mendes`.

## Estrutura
- `fundamental_i/`
  - arquivos do `1o` ao `5o ano`
- `fundamental_ii/`
  - arquivos dos anos finais por componente/area
